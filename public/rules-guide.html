EmbroMine

EmbroMine — Minecraft сервер для спонсоров. Доступ предоставляется в виде бонуса за подписку в Boosty, а не покупается!

Проект создан в формате "от спонсора спонсорам". Каждый участник может вступить в состав администрации, доказав ответственность и уважение к сообществу.

Пошаговая инструкция

1

Получаем доступ

Доступ к проекту имеется при наличии роли спонсорства Boosty в Discord

  • ✓ Подпишитесь на Boosty
  • ✓ Получите роль спонсора в Discord
  • ✓ Получите доступ к серверу
2

Вход на сервер

Для игры у вас должен быть установлен Minecraft и вы должны быть авторизованы в вайтлисте

  • ✓ Установите Minecraft
  • ✓ Авторизуйтесь в вайтлисте
  • ✓ Подождите одобрения администрации
3

Играем

Соблюдаем правила, уважаем друг друга и вместе развиваем проект

  • ✓ Ознакомьтесь с правилами
  • ✓ Уважайте остальных игроков
  • ✓ Развивайте сообщество
⚠️
Важно!

Пункты не пропускать. Выполнять каждый пункт последовательно!
